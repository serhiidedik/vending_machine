# Vending machine

The **Vending Machine** is a classic application often used to demonstrate core software engineering principles, such as Object-Oriented Programming (OOP), state management, and transaction integrity.

This project simulates a real-world vending machine: handling product selection, processing payments, calculating change, and managing inventory.
